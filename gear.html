My Daily Setup

Góc công nghệ mình đang dùng để học code, Linux và build project.

Không phải setup đắt nhất, nhưng đây là combo đủ thực dụng: màn rộng để đọc code, laptop để thử nghiệm, bàn phím gõ lâu thoải mái, chuột nhẹ và tai nghe rõ chi tiết.

Thiết Bị

Những món mình đang sử dụng

Display

Philips 271V8

Màn hình 27 inch dùng để đọc tài liệu, chia đôi cửa sổ code và terminal. Điểm mạnh là không gian hiển thị rộng, hợp học lâu và làm việc đa nhiệm.

  • Không gian rộng cho code + docs
  • Phù hợp workflow nhiều cửa sổ
  • Setup gọn cho bàn học
Main Machine

MSI GF63 9RCX

Máy chính để học lập trình, chạy Linux, thử tool mới và làm web. Thiết kế gaming mỏng giúp dễ mang theo nhưng vẫn đủ lực cho nhu cầu học tập.

  • Máy chính cho C++ và web
  • Phù hợp thử Linux/workflow
  • Dùng để build và test project
Keyboard

AULA AU75

Bàn phím gõ hằng ngày, layout gọn nhưng vẫn đủ phím cần thiết. Hợp viết code, chat, ghi chú và dùng shortcut thường xuyên.

  • Layout gọn, tiết kiệm diện tích
  • Cảm giác gõ phù hợp dùng lâu
  • Dễ hợp với setup tối giản
Mouse

Lingbao M5 Pro Max

Chuột nhẹ, phản hồi nhanh, hợp thao tác nhiều giữa editor, browser và terminal. Một món nhỏ nhưng ảnh hưởng khá nhiều đến cảm giác làm việc.

  • Di chuyển nhanh, gọn tay
  • Hợp thao tác liên tục
  • Phối tốt với bàn phím compact
Audio

KZ EDX Pro X

Tai nghe in-ear để nghe nhạc, xem video học và tập trung khi code. Gọn, dễ mang theo, đủ rõ để dùng trong nhiều tình huống hằng ngày.

  • Gọn nhẹ, tiện mang theo
  • Phù hợp học và giải trí
  • Cách âm tốt hơn loa laptop
Workflow

Cách mình dùng setup này

Màn hình ngoài dành cho tài liệu, browser và terminal phụ. Laptop là trung tâm xử lý. Bàn phím, chuột và tai nghe giúp mọi thao tác hằng ngày nhanh hơn, tập trung hơn.

Mở editor và terminal trên laptop.

02

03

04

Đeo KZ EDX Pro X để tập trung học và debug.
